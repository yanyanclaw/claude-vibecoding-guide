# Claude Vibe Coding 指南

給軟體業 PM、設計師與非工程師的 AI 開發上手書

## 關於本書

為什麼跟 AI 說了半天，它還是在那邊繞圈圈修不好？因為你缺的不是更好的 AI，而是「怎麼跟 AI 協作」的方法論。

本書帶你從 Claude Artifacts 入門，到 Claude Code 進階，拿回「原型自主權」。
